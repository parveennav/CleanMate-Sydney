## ğŸ†˜ Common Issues & Solutions

### Issue: Cards not showing hover effect
**Solution**: Check if Font Awesome CSS is loaded (for icons)

### Issue: Layout looks broken on mobile
**Solution**: Ensure viewport meta tag is in HTML:
```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

### Issue: Animations too fast/slow
**Solution**: Adjust transition timing in CSS variables

### Issue: Colors don't match brand
**Solution**: Update CSS custom properties in `:root` selector
